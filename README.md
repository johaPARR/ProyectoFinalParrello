# Proyecto Final - JavaScript  
## Tienda Araperful  

### 📌 Descripción  
Este proyecto consiste en un simulador interactivo de e-commerce desarrollado en JavaScript.  
Permite al usuario visualizar productos, agregarlos al carrito, gestionar cantidades, seleccionar un método de pago y finalizar una compra simulada.

El flujo completo de compra incluye:
- Visualización de productos desde un archivo JSON (mediante Fetch)
- Agregar productos al carrito
- Persistencia de datos con LocalStorage
- Cálculo dinámico del total
- Selección de método de pago
- Aplicación de descuento en efectivo
- Confirmación final de compra

---

### 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- LocalStorage

---

### 📚 Librerías utilizadas

1. **SweetAlert2**  
   - Versión: 11  
   - Uso: Alertas personalizadas para confirmaciones y mensajes al usuario  
   - Documentación: https://sweetalert2.github.io/

2. **Toastify JS**  
   - Uso: Notificaciones animadas al agregar productos al carrito  
   - Documentación: https://apvarun.github.io/toastify-js/

3. **Luxon**  
   - Versión: 3.4.4  
   - Uso: Manejo y formateo de fechas en la confirmación de compra  
   - Documentación: https://moment.github.io/luxon/

✔ Carga de productos mediante Fetch y archivo JSON  
✔ Renderizado dinámico del DOM
